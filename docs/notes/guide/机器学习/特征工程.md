# 特征工程

## 一、什么是特征工程

**特征工程**是指：从原始数据中，通过**清洗、提取、转换、构造、筛选**，生成高质量、有区分度、适合模型学习的特征的全过程。

> **核心价值**：数据和特征决定模型上限，算法只是逼近这个上限。
> **目的**：减少噪声、降低冗余、简化模型、提升精度、防止过拟合、加快训练速度。

特征工程整体包含四大模块：

1. **特征提取**：从原始数据挖掘有效信息
2. **特征预处理**：归一化、标准化、缺失值处理、异常值处理
3. **特征构造/衍生**：组合特征、统计特征、时间特征
4. **特征选择**：剔除无用、冗余、噪声特征

---

## 二、特征选择 核心概念

特征选择：从**已有特征集合**中，筛选出最优子集，**不创造新特征**，只做筛选。

**解决问题**：维度灾难、特征冗余、多重共线性、无关特征干扰。

### 三大主流分类：过滤法、包裹法、嵌入法

#### 1. 过滤法（Filter）

先筛选特征，再训练模型，**速度快、泛化强、复杂度低**，适合高维数据。

**原理**：依靠**统计指标、相关性**评估特征，和模型无关。

##### 常用方法

1. **方差筛选**
    - 剔除方差极小/方差为0的特征（取值几乎无变化，无区分能力）
    - 例：全是0/全是1的列直接删除

2. **相关系数筛选**
    - 皮尔逊相关系数：评估**线性相关**
    - 流程：计算特征与标签相关性，设定阈值，保留高相关特征；剔除特征间高相关冗余特征

3. **卡方检验**
    - 适用于：**离散特征+分类任务**
    - 衡量特征与目标变量的关联性，卡方值越大，区分能力越强

4. **互信息**
    - 捕捉**非线性关联**，弥补皮尔逊只能看线性的缺陷
    - 适合复杂分布数据

5. **ANOVA方差分析**
    - 适用于：**连续特征+分类任务**
    - 判断不同类别下特征分布是否有显著差异

---

#### 2. 包裹法（Wrapper）

以**模型最终效果**为目标，不断筛选组合特征，用模型误差评判特征好坏。

- **优点**：效果最优、贴合具体模型
- **缺点**：计算量大、容易过拟合、耗时长

##### 包裹法常用方法

1. **递归特征消除 RFE**
    - 逻辑：先全量特征训练，逐步剔除权重最小的特征，迭代直到指定特征数
    - 配合：线性模型、SVM、树模型

2. **贪心搜索系列**
    - 向前选择：从0个特征开始，逐个加入提升效果最好的特征
    - 向后消除：从全量特征开始，逐个删掉影响最小的特征
    - 双向搜索：结合前两者

3. **遗传算法、随机搜索**
    - 启发式搜索，适合超高维、非线性复杂场景

---

#### 3. 嵌入法（Embedded）

**训练模型的同时自动完成特征选择**，融合过滤与包裹优点，效率+效果平衡。

##### 嵌入法常用方法

1. **L1 正则化（Lasso）**
    - 核心：产生**稀疏解**，压缩大量无用特征系数为0，自动剔除无效特征
    - 适用：线性回归、逻辑回归

2. **L2 正则化（Ridge）**
    - 不会置零系数，只会缩小权重，侧重降共线性，弱选择能力

3. **弹性网 Elastic Net**
    - 结合L1+L2，解决高维特征、特征共线性问题

4. **树模型自带特征重要性**
    - 随机森林、XGBoost、LightGBM、CatBoost
    - 依据：基尼系数、信息增益、分裂次数计算特征重要度
    - 直接设定阈值，删除低重要性特征

---

## 三、补充：实操场景选用建议

1. **数据维度极高、数据量大** → 过滤法（方差、卡方、相关系数）
2. **追求模型精度、算力充足** → 包裹法（RFE）
3. **常规训练、兼顾效率与效果** → 嵌入法（Lasso、树特征重要性）
4. **存在多重共线性** → 相关系数筛选 + L2/弹性网

---

## 四、一句话极简总结

1. **特征工程**：把原始数据变成模型能高效学习的优质特征的整套流程。
2. **特征选择三类**：
    - 过滤法：靠统计指标筛，快；
    - 包裹法：靠模型效果筛，准；
    - 嵌入法：训练时自动筛，均衡。
