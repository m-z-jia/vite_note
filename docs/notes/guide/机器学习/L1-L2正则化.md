# 归纳整合：L1、L2 范数 + 正则化

## 一、L1、L2 范式（范数）定义

范式即向量长度/大小的度量，设权重向量：$w = (w_1, w_2, ..., w_n)$

### 1. L1 范数（1-范数）

$$||w||_1 = \sum_{i=1}^n |w_i|$$

向量所有元素**绝对值之和**，几何轮廓为菱形。

### 2. L2 范数（2-范数/欧氏范数）

$$||w||_2 = \sqrt{\sum_{i=1}^n w_i^2}$$

向量元素平方和再开根号，就是欧式距离，几何轮廓为圆形；
正则化中常用**平方L2范数**$\sum w_i^2$，方便求导计算。

### 举例

$w = (3, -4)$

$||w||_1 = 3 + 4 = 7, ||w||_2 = \sqrt{3^2 + (-4)^2} = 5$

## 二、L1、L2 正则化公式

总损失函数通用形式：

$$Loss_{总} = Loss_{原} + 正则惩罚项$$

### 1. L1 正则（Lasso）

$$Loss = Loss_{原} + \lambda \sum |w_i|$$

### 2. L2 正则（Ridge）

$$Loss = Loss_{原} + \lambda \sum w_i^2$$

### 公式两项含义

- 右边第一项 $Loss_{原}$：**原始损失**
  衡量模型预测值与真实值的误差，作用是让模型拟合训练数据。
- 右边第二项（带$\lambda$）：**正则惩罚项**
  $\lambda$ 为正则强度，用来限制权重大小、降低模型复杂度、防止过拟合。

## 三、L1 与 L2 核心区别

### 1. L1正则

权重易被压缩为 0，产生稀疏解，自带特征选择、剔除冗余特征；优化计算稍复杂。

### 2. L2正则

权重只会无限趋近 0 不会归零，全部特征保留；对噪声更稳定，梯度计算简单，深度学习常用。

## 四、一句话总结

1. L1范数：绝对值求和；L2范数：平方和开根号。
2. 损失第一项管拟合误差，第二项管防止过拟合。
3. L1 选特征做减法，L2 缩权重稳模型。
