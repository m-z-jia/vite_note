# 实验4 App 运行说明

## 一、App 简介

本次实验将实验4中的 Java 程序整理成一个可直接下载运行的桌面应用。应用使用 Java Swing 实现图形界面，并把相关运行环境一起打包，下载后不需要再单独配置 JDK。

App 中主要包含两个功能模块：

1. **算术练习**：生成加、减、乘、除等四则运算题目，支持输入答案并判断结果。
2. **扫雷小游戏**：提供经典扫雷玩法，点击格子进行探索，避开隐藏的地雷。

## 二、下载链接

[点击下载 Lab4App.zip](/downloads/java-lab4/Lab4App.zip)

> 压缩包内已经包含程序文件和运行时环境，解压后即可运行。

## 三、运行方式

下载并解压 `Lab4App.zip` 后，进入解压得到的 `Lab4App` 目录，根据系统环境选择运行方式：

```bash
runtime/bin/java -jar app/Lab4App.jar
```

如果在 Windows 文件资源管理器中操作，也可以进入 `Lab4App/app` 目录，双击 `Lab4App.jar` 运行；如果双击无法启动，则使用上面的命令行方式运行。
